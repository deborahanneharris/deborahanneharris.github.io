Where are the Grandkids?

July 18, 2015

I had a brain infarction and couldn’t think of a “real world object” to use as an example of “Class”. Then my friend Jennifer Poeng suggested that I read Erik Trautman’s article about classes.

His use of Vikings as a class inspired me to think of my five (and counting) grandchildren.

The “class” method is a way to organize data among various different methods. I have set up a class named Grandchildren. My grandchildren have many different attributes to record but they all share the same basic information. I was interested in tracking their Full Name, Sex, Date of Birth, Favorite Activity, Favorite Food so I could keep track of their birthdays, potential recreational activities and seasonal gifting opportunities.

First, we would name the class Grandchildren and then #initialize the data that we want to track about my Grandchildren class (Full Name, Sex, Date of Birth, Favorite Activity, Favorite Food).

Next, we will set up individual instance variables that all class Grandchildren share but they have different values per individual. For example: “Mark Darrell Harris” has a different Full Name instance variable value than “Derek Oscar Fuchs” but they BOTH have a Full Name variable. 

class Grandchildren
  def initialize(full_name, sex, date_of_birth, favorite_activity, favorite_food)
    @full_name = full_name
    @sex = sex
    @date_of_birth = date_of_birth
    @favorite_activity = favorite_activity
    @favorite_food = favorite_food
  end
end

Now how would an instance method apply to this example? What kind of activities or methods could we have class Grandchildren do?

We could define a method for birthday party and list the appropriate instance variables. Since these methods are being called or used by class Grandchildren they are called instance methods. 

def party(full_name, date_of_birth, favorite_activity, favorite_food)
      #code for party preparations
    end

In conclusion, this was a great topic to incorporate my interest in my grandchildren and also organize their disparate data. As I become more adept at setting up classes I will definitely start tracking my grandchildren in this manner. I'll be a #Dev Grandma for my grandkids.
